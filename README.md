# BlazeSpawnerLock
Simple minecraft paper 1.21.x plugin that locks the modification of blaze spawners by the players to keep SMPs alive
## Configurable Settings
This plugin offers a configurable message displayed when a player tries to break a blaze spawner
## Compatibility
Version: **1.21.x** (wider support planned)<br>
**Bukkit** ✅
**Spigot** ✅
**Paper**  ✅
**Purpur** ✅
**Folia**  ❌ (planned)
## Permissions
**blazespawnerlock.bypass** - bypass the plugin and allow players with the permission to break spawners (default: op)<br>
**blazespawnerlock.reload** - reload the plugin configuration (default: op)
## Commands
**/blazespawnerlock reload** - reload the plugin configuration<br>
**/blazespawnerlock update** - command to check for update<br>
**/bsl** - command alias
